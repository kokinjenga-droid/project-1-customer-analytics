# Project 1: Customer & Revenue Analytics

## Overview

This project analyses customer purchasing behaviour using the Online Retail dataset.

The goal was to generate recommendations around:

- Customer retention
- Revenue growth
- Customer segmentation

Tools used:

- SQL
- Power BI
- GitHub

## Key Metrics

- Monthly Repeat Purchase Rate
- RFM Segmentation
- 6-Month Cohort Retention
- Revenue by Customer Segment
- At-Risk Customer Analysis

## Key Findings

- Revenue is concentrated among Champions and Loyal customers.
- Cohort retention declines significantly after acquisition.
- At-Risk customers represent a major revenue recovery opportunity.

## Recommendations

- Retain Champion customers through loyalty initiatives.
- Upsell Loyal customers.
- Launch targeted win-back campaigns for At-Risk customers.
